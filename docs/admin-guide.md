# Администрирование бота

## Команды админа (в чате бота)

| Команда | Что делает |
|---------|-----------|
| `/users` | Показать список разрешённых пользователей |
| `/adduser 123456789` | Добавить пользователя |
| `/removeuser 123456789` | Удалить пользователя |
| `/myid` | Узнать свой Telegram ID |

## Как добавить нового человека

1. Дай ему ссылку на бота
2. Он пишет боту что угодно — получает сообщение "нет доступа" со своим ID
3. Тебе приходит уведомление с его именем и готовой командой `/adduser`
4. Нажимаешь на команду — готово
